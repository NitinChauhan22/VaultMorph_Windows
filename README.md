# 🔐 VaultMorph - Secure Offline Password Manager for Windows

**100% Offline • Zero-Trust • AES-256-GCM Encryption**

VaultMorph is a privacy-first password manager designed to securely store your credentials and sensitive data locally using industry-standard encryption.

No cloud. No telemetry. Full control.

## ✨ Features

* 🔒 AES-256-GCM encryption with Argon2id key derivation
* 💾 100% offline vault (no cloud sync)
* 🔐 Zero-trust architecture (everything encrypted at rest)
* 📝 Secure notes storage
* 🔑 Built-in TOTP 2FA authenticator
* 📤 Encrypted backup and restore (.vmb format)
* 📊 Password health analysis (weak/reused passwords)
* 🚀 Fast and lightweight native Windows performance

---

## 🔒 Security Overview

VaultMorph uses industry-standard cryptographic practices:

* AES-256-GCM encryption for all sensitive data
* Argon2id memory-hard key derivation
* Encrypted local storage (SQLite)
* Master password is never stored in plaintext
* Automatic memory cleanup on vault lock

Your data never leaves your device.

---

## ⚠️ Important Notes

* Your master password **cannot be recovered** if lost
* Always maintain secure backups of your vault
* This application works **completely offline by design**

---

## 🚧 Limitations

* Windows platform only
* No cloud synchronization
* No browser extension support
* Free edition limited to 100 password entries
